#웹 OCR 근로계약서 인식 프로젝트
이 프로젝트는 근로계약서를 OCR(광학 문자 인식) 기술을 사용하여 스캔하고 텍스트로 추출하고, 해당 내용에 대한 자문을 제공하는 애플리케이션입니다. 이를 통해 사용자는 근로계약서의 내용을 빠르게 확인하고 저장할 수 있습니다.
